= csv_to_sql_parser

== DESCRIPTION:

* From a csv file, I want to be able to generate SQL inserts, 
  it may be useful when trying to import data from csv files.

== FEATURES/PROBLEMS:

* It is still pretty simple. But it's easy ;)

== SYNOPSIS:

  See specs. 

== REQUIREMENTS:

* RSpec for tests

== INSTALL:

* Not yet :)

== DEVELOPERS:

After checking out the source, run:

  $ rake newb

This task will install any missing dependencies, run the tests/specs,
and generate the RDoc.
